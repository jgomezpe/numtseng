We have a problem when we wish to find an object (called it unknown object) from a group of objects (called known objects) and a relation that the unknown object maintains with the known objects.

Solving a problem is to find the unknown object. In Numtseng, instead of just solving problems, we will learn a process to solve problems. To solve a problem we must:
  • Characterize the known objects in a precise manner, that means, we must answer the following question: ¿What are the known objects?. For doing this, we need to anwer the following questions:
    • ¿Which are the types of the known objects?
    • ¿What are the values such known objects can take?
  • Characterize the unknown object in a precise manner, that means, we need to anwer the following questions:
    • ¿Which is the type of the unknown object?
    • ¿What are the values such unknown object can take?
  • Answer, in a precise manner, ¿How are related the known objects to the unknown object?
