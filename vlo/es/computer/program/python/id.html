Un identificador es una secuencia de símbolos que utilizamos como nombre para las variables, funciones, arreglos, clases y otras estructuras propias de los lenguajes de programación. Los identificadores en Python se escriben como secuencias de caracteres alfanuméricos (letras del abecedario inglés o los dígitos) o el guión bajo (_), secuencias en las que el primer símbolo no puede ser un dígito y que deben tener al menos una letra. Los identificadores en Python deben tener una longitud mayor o igual a uno.

Adicionalmente, un identificador no debe ser una palabra reservada del lenguaje. En Python las palabras reservadas son las siguientes:
  • and, as, assert, break, class,
  • continue, def, del, elif, else,
  • except, finally, False, for, from
  • global, if, import, in, is,
  • lambda, nonlocal, None, not, or,
  • pass, raise, return, True, try,
  • with, while, yield
