Python es un lenguaje de programación interpretado (esto indica que ejecuta linea por linea el código), que fue diseñado por Guido van Rossum con el fin de facilitar la lectura de código fuente. Python se considera un lenguaje multi-paradigma pues soporta parcialmente la programación orientada a objetos, la programación imperativa y la programación funcional.

La escuela de Python, en Numtseng, es un curso en línea para aprender los fundamentos del lenguaje de programación Python.

El contenido del curso está organizado como un libro: capítulos, secciones y finalmente sesiones que pueden ser navegadas mediante la tabla de contenidos ubicada a la izquierda (initialmente oculta en pantallas de tamaño pequeño) o usando los botones en la barra de navegacion:

para ir al contenido hermano previo (contenido anterior al mismo nivel).

para ir al contenido del nivel padre (nivel que contiene la unidad).

para ir al primer contenido hijo (primer contenido desarrollado en el nivel).

para ir al siguiente contenido hermano (siguiente contenido en el mismo nivel).

El contenido de la sesión se presenta en un componente enriquecido con matemáticas y se usa el ambiente de programación https://replit.com para escribir y probar código ejemplo. Para ver el código, es posible que debamos seleccionar Files y luego el archivo main.py en el ambiente replit.

De esta manera, el programa :
  • Imprime en consola el texto Hola Mundo (línea 1),
  • Solicita al usuario que ingrese un nombre (línea 2),
  • y finalmente muestra el nombre leido acompañado de un saludo (línea 3).
