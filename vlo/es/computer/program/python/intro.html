¡¡Bienvenido a la escuela de Python de Numtseng!!.

En este curso aprenderemos los fundamentos del lenguaje de programación Python, el cual fue diseñado por Guido van Rossum con el fin de facilitar la lectura de código fuente. Python es considerado un lenguaje multi-paradigma pues soporta parcialmente la programación orientada a objetos, la programación imperativa y la programación funcional. A diferencia de lenguajes como C o Pascal, Python es un lenguaje interpretado, esto es, se va ejecutando linea por linea el código.

El contenido del curso se presenta en un componente enriquecido con matemáticas y se usa el ambiente de programación replit, disponible en la dirección https://replit.com, para escribir y probar código ejemplo. Para ver el código, es posible que debamos seleccionar Show Files y luego el archivo main.py en el ambiente replit.

Por ejemplo, en el programa:
  • Se imprime en consola el texto Hola Mundo (línea 1),
  • Se solicita al usuario que ingrese un nombre (línea 2),
  • y finalmente se muestra el nombre leido acompañado de un saludo (línea 3).
Para ejecutar el programa debemos seleccionar Run en el ambiente replit y usar la consola para interactuar con el programa.

El curso está organizado en capítulos como un libro, los capítulos se desarrollan en secciones, las secciones en unidades y las unidades en sesiones. Podemos navegar el curso mediante la tabla de contenidos ubicada a la izquierda (inicialmente oculta en pantallas de tamaño pequeño) o mediante los botones en la barra de navegacion:

para ir al tema que precede al actual y que esta al mismo nivel.

para ir al tema general que contiene al actual.

para ir al primer tema específico del actual.

para ir al tema que sigue al actual en el mismo nivel.

Podemos seguir el contenido como una presentación dinámica si un botón de tocar aparece en la parte superior izquierda de la página.
