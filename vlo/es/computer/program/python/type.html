Un tipo de dato es un conjunto de valores (objetos) que pueden tomar las variables y constantes en Python.

Un conjunto \(A\) es una colección 'bien-definida' de objetos. Una colección se dice bien-definida si es posible determinar de manera precisa tanto los objetos que están en la colección como los que no están. Los objetos que pertenecen a un conjunto \(A\) son llamados elementos de \(A\). En general, usamos \(x \in A\) para decir que el objeto \(x\) es un elemento de \(A\) y \(x \notin A\) para decir que el objeto \(x\) no es un elemento de \(A\). Si quieres saber más de conjuntos sigue este enlace que trata el tema de conjuntos en Numtseng: La Escuela de Matemáticas.

Como en la mayoría de lenguajes de programación, en Python una variable o constante puede ser de un tipo de dato primitivo, es decir, de uno de los conjuntos de valores básicos con los que cuenta Python, o de tipo no-primitivo, es decir, que toma un valor que se construye o compone a partir de tipos de datos primitivos y/o no primitivos. En esta sección solamente estudiaremos los primitivos, los no-primitivos los estudiaremos más adelante.

Los cuatro tipos de datos primitivos en Python son:
  • int: Una aproximación a los enteros (\(\mathbb{Z}\)),
  • float: Una aproximación a los reales (\(\mathbb{R}\)),
  • boolean: Los valores de verdad (aquí los notaremos \(\mathbb{B}\)), y consta de los valores verdadero True y falso False, y
  • string: Las cadenas de caracteres o textos.
