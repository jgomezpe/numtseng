Una variable es un espacio de memoria donde almacenamos un dato o valor. Es un espacio donde guardamos la información necesaria para realizar las acciones que ejecutan nuestros programas.

Variable \(x\) en la memoria \(\mathcal{T}\)

En Python, como en todo lenguaje de programación, una variable debe ser de algún tipo de dato, es decir, de uno de los conjuntos de valores básicos con los que cuenta. Python no requiere que declaremos (definamos) el tipo de dato de una variable de manera explícita, pues maneja los tipos de datos de manera implícita, esto es, infiere el tipo de dato de una variable de acuerdo a lo que tiene almacenado.

Si queremos declarar una variable \(x\) requerimos un identificador para ella, su tipo de dato (que es opcional en Python), y un valor inicial para ella, declaración que escribimos \(x : T valor\). A un valor concreto de un tipo de dato que podamos escribir en Python lo llamamos literal de ese tipo de dato.
