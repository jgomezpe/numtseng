En este curso aprenderemos los fundamentos del lenguaje de programación Python. Python es un lenguaje de programación diseñado por Guido van Rossum con el fin de facilitar la lectura de código fuente. Python es interpretado, esto es, ejecuta linea por linea el código. Además, Python se considera un lenguaje multi-paradigma pues soporta parcialmente la programación orientada a objetos, la programación imperativa y la programación funcional.

El contenido se presenta en un componente enriquecido con matemáticas y se usa el ambiente de programación https://replit.com para escribir y probar código ejemplo. Para ver el código, es posible que debamos seleccionar Show Files y luego el archivo main.py en el ambiente replit.

Por ejemplo, el programa:
  • Imprime en consola el texto Hola Mundo (línea 1),
  • Solicita al usuario que ingrese un nombre (línea 2),
  • y finalmente muestra el nombre leido acompañado de un saludo (línea 3).
El curso está organizado en capítulos como un libro, los capítulos se desarrollan en secciones, las seciones en unidades y las unidades en sesiones.
