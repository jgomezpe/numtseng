En Python, para describir el conjunto de booleanos, es decir, el conjunto de los valores de verdad $\left{F,V\right}$ se usa el literal bool, para el valor de verdad falso (F) se usa el literal False y para el valor de verdad verdadedro (V) se usa el literal True. Note que los literales para falso y verdadero tienen su primer letra mayúscula. Si quieres saber más del conjunto de los valores de verdad (o booleanos) sigue este enlace que trata el tema del conjunto de valores de verdad (que aquí notamos $\mathbb{B}$) en Numtseng: La Escuela de Matemáticas.

Si queremos que una variable $b$ sea de tipo booleano, es decir $b \in \mathbb{B}$, en Python la declaramos como b:float. Cuando declaramos una variable debemos iniciarla con algún valor. Por ejemplo, en la línea 1 del programa, no declaramos el tipo de la variable b pero la iniciamos en falso False, por lo que implícitamente la estamos declarando de tipo booleano. En la línea 2 estamos declarando a la variable y de tipo booleano y le estamos dando un valor inicial de verdadero True. En la línea 3, estamos declarando la variable z de tipo booleano y le estamos dando un valor inicial de falso False.
