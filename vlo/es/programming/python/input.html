La función input nos permite leer valores ingresados por el usuario desde la consola. Estos valores los recibe la función input como una cadena de caracteres apenas el usuario presione la tecla enter ($\hookleftarrow$). Es posible enviarle un mensaje al usuario para que sepa que esta ingresando con la funcion input.

Por ejemplo, en la línea 1 del programa, se le muestra al usuario el mensaje "Tu nombre?" y se le pide que ingrese un valor, el cual es mostrado en la línea 2 antecedido por el texto "Un gusto".

Si queremos leer un número entero (sin parte decimal), aplicamos a la cadena leida la función int, ver línea 3. Si el usuario ingresa un valor que no corresponda con un entero nos generará un mensaje de error, pero si ingresa un valor que representa un entero, podremos operar dicho valor como en la suma que hacemos en la línea 4, y cuyo resultado podemos ver con el print de la línea 5.

Si queremos leer un número real (con parte decimal), aplicamos a la cadena leida la función float, ver línea 6. Si el usuario ingresa un valor que no corresponda con un real nos generará un mensaje de error, pero si ingresa un valor que representa un real, podremos operar dicho valor como en la división que hacemos en la línea 7, y cuyo resultado podemos ver con el print de la línea 8.
